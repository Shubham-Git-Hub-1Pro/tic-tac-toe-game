# 🎮 Tic Tac Toe Game

A fully responsive and interactive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.  
The project features a clean UI, smooth animations, score tracking, and works perfectly on both **desktop and mobile devices**.

🔗 **Live Demo:**  
👉 https://chic-blini-9042da.netlify.app/

---

## ✨ Features

- ❌⭕ Interactive Tic Tac Toe gameplay (X vs O)
- 🎯 Automatic win detection with animated winning line
- 📊 Score counter for:
  - X wins
  - O wins
  - Draws
- 🔄 Reset button to restart the game
- 💡 Turn indicator with animation
- 📱 Fully responsive design (mobile & desktop)
- 🎨 Eye-friendly UI with modern gradient design
- 🚀 Deployed using **Netlify**

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling, animations & responsiveness  
- **JavaScript (Vanilla JS)** – Game logic & DOM manipulation
